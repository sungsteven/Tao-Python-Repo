This project is about a DNP3 master simualtor. The following features are included in this version (will be expanded if needed in the future)
1. Connection to a DNP3 outstation via Serial, UDP or TCP mode
2. Support both solicited and unsolicited responses
3. Support all standard SCADA objects used for reporting input data and issuing controls
4. Use treeview to display octet values of data together with interpretations
5. Automatically handle time sync, application confirm, user data acknowledgement and unsolicited messages
